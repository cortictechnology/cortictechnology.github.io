Getting Started with the Cortic Platform

Welcome to the Cortic Platform. This guide will help you set up and familiarize yourself with the Cortic ecosystem's vital components across macOS, Windows 11, and Ubuntu Linux.

1. Components of the Cortic Platform

The Cortic platform consists of:

  • Runtime Library: Executes core functions.
  • SDK (Software Development Kit): Offers tools for application development.
  • Device Manager: Manages and integrates devices.
  • Cortic AI Hub: The main gateway for platform access.
  • Cortic App Store: A repository of platform-specific applications.

The Cortic AI Hub is the essential component for users.

2. Pre-installation Steps

Before installing the Cortic AI Hub:

  1. System Requirements Check: Ensure compatibility with your OS.
  2. Network Configuration: Secure a stable internet connection.
  3. Required Dependencies:
    • macOS: Install any pending software updates from the Apple Menu > System Preferences > Software Update.

    • Windows 11: Ensure your system is up-to-date by going to Settings > Update & Security > Windows Update.

3. Downloading the Cortic AI Hub

Navigate to the official Cortic website or repository. Download the respective Cortic AI Hub package for your OS:

  • macOS: Download the .dmg file.

  • Windows 11: Download the .exe file.

  • Ubuntu Linux: Download the .deb or .tar.gz file.

4. Installing the Cortic AI Hub

For each operating system:

  • macOS:
    1. Open the .dmg file.
    2. Drag the Cortic AI Hub to the Applications folder.
    3. Eject the installation disk image.
      <screenshot>
  • Windows 11:
    1. Double-click on the .exe file.
    2. Follow on-screen instructions.
      <screenshot>
  • Ubuntu Linux:
    1. For .deb files, use sudo dpkg -i filename.deb in the terminal.
    2. For .tar.gz files, extract the archive and follow the included README.
      <screenshot>
